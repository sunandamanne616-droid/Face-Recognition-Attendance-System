# 🎯 Face Recognition Attendance System

An AI-powered attendance management system that uses **face recognition** to automatically mark attendance, reducing manual effort and improving accuracy.

---

## 🚀 Features

* 👤 Real-time face detection using webcam
* 🧠 Face recognition using machine learning models
* 📋 Automatic attendance marking
* 🗂️ Attendance stored in structured format (CSV/Database)
* ⚡ Fast and efficient processing
* 🔒 Reduces proxy attendance

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries Used:**

  * OpenCV
  * NumPy
  * face_recognition / dlib
* **Tools:**

  * VS Code
  * Git & GitHub


## 📸 How It Works

1. Capture face data and store it in the dataset
2. Train the model on collected images
3. Detect faces in real-time using webcam
4. Recognize faces and mark attendance automatically
